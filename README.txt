DarkSky API only gets 1000 free calls a day. Therefore, the function runs every 1.44 minutes. You must wait 
until the data appears, then it will update periodically.

The landscape clock is designed for the dorm lobbies, giving a more detailed forecast for the student about to leave their
dorm.

The portrait clock is designed for basically everywhere else, with just basic weather info as well as a daily bible
reference for the christian campus. The verse changes whenever the page is refreshed.
